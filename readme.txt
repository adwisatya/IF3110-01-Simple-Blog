Panduan menggunakan simple-blog-of-bangsatya

1. Buat database simpleblog
2. Import file simpleblog.sql
3. Pindahkan folder simpleblog ke direktori default web server (htdocs untuk xampp, /var/www/html untuk apache)
4. Akses menggunakan web browser
5. Simpleblog siap digunakan.
